# Task Manager App

this is a task manager app with role based access control

## how to run

### backend
```
cd server
npm install
npm run dev
```

### frontend
```
cd client
npm install
npm run dev
```

## features
- user can register and login
- admin and user roles
- user can create, update, delete their own tasks
- admin can see all users and all tasks
- admin can delete any user or task
- admin can change user status (active/inactive)
- activity logs for login and task actions
- admin dashboard with stats
